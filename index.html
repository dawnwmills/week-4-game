Crystals Collector

How to Play

  • You will be given a random number at the start of the game.
  • There are 4 crystals, each with a different value.
    • RED
    • GREEN
    • PURPLE
    • YELLOW
  • Click a crystal to add points to your score.
  • Match the random number to WIN the game!
  • Score above the random number and you LOSE.
  • The value of each crystal is hidden from you until you click on it.
  • The value of each crystal changes at the start off each game!
